# Intro
- Create Lead Qualification Questions.
- AI-Driven Chatbot Engages & Collects Responses.
- AI Analyzes & Scores Leads.

## Tech Stack

- NextJS (React)
- NestJS
- Supabase (Default to PostgreSQL)
- Authentication (Clerk)
- AI model (LLAMA-3.1 70B)
